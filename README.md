# Analysis and plotting for AI safety and ethics review 

## AI safety

To run the code in this folder please clone this repo and make sure you have all requirements installed. 
You can run the following command from the to install the requirements automatically:
```bash
pip install ./requirements.txt
```

To produce the different plots in the paper, you can uncomment various sections of the `code/analysis.py` file.
Currently, the file is set to create topic modeling visualisations using BERTopic.



## AI ethics

To produce the plots in the AI ethics paper, you can follow the Jupyter notebook under `AI Ethics\analysis.ipynb`.
